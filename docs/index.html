PyGMT 範例：全球地形立體視圖

這是使用 PyGMT 輕鬆繪製的全球地球浮雕圖（Earth Relief），採用正交投影（Orthographic Projection），呈現地球如球體般的立體視角。
圖中藍色代表海洋深處，綠色為平原，棕色為高山，並加入陰影效果（shaded relief）讓地形更立體生動。
只需幾行 Python 程式碼，就能載入 GMT 內建的全球地形資料（如 @earth_relief），並使用 grdimage 與 geo 色盤快速生成類似專業級圖片！

推薦影片：Quick Introduction to Plotting with PyGMT - AGU 2021

影片描述： 這是為 AGU 2021 秋季會議創作的 PyGMT 繪圖快速介紹。查看 GitHub 上的 Jupyter notebook：連結。

主要內容：

  • PyGMT 是一個開源 Python 庫，用於地理空間處理、分析和可視化。
  • 介紹使用 Figure 類別創建地圖，這是 PyGMT 繪圖的核心。
  • 創建簡單基底地圖：
    • 導入 PyGMT 模組。
    • 使用 Figure 類別創建地圖實例。
    • 使用 basemap 方法設定地圖參數： region（定義邊界，例如經度 -180 到 180，緯度 -70 到 70）、projection（指定投影，例如 Mercator 寬 20 cm）、frame（控制邊界外觀，設定為 True 以自動設定）。
    • 使用 show 方法顯示地圖。
  • 添加海岸線內容：
    • 使用 coast 方法繪製地理特徵。
    • 自訂外觀，例如陸地顏色（深綠）和水域顏色（淺鋼藍）。
    • 使用快捷方式如 d 為默認全球區域，h 為 Hammer 投影。
  • 繪製柵格數據：
    • 使用 grdimage 方法繪製柵格數據集。
    • 存取遠端數據集（例如 SRTM15+ 地球地形網格，20 弧分解析度）。
    • 應用自動陰影並使用 show 顯示。

額外資源： PyGMT 文件、教程、圖庫範例和 Q&A 論壇，用於進一步學習。注意：影片未涵蓋 PyGMT 或 GMT 的安裝。
